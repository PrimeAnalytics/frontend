Thanks & Support

IMPORTANT: This is NOT a Wordpress theme and you cannot install this with Wordpress. This is a HTML5 template and requires HTML knowledge.

Thanks

Firstly, thank you for purchasing Skyline. We hope you enjoy using the template.

Support

Please contact support via the support tab of the item that you purchased.

For questions

If you have any problems or questions that's not been covered please feel free to contact support, but do please check your question isn't covered by the documentation provided first.

Please be aware support doesn't cover customization requests and is for questions on usage and bugs only.

Before reporting issues

When contacting support about a problem, please give a detailed description of the problem you are experiencing, when the problem occurs and if there were any preciptating events.

Please provide as much detail as possible to make it easier for us to understand the issue and help you further.

Providing information

Please provide a link if at all possible or a screenshot it really helps us to understand and it some cases we can't help without a link. (In some instances sending a copy of a page is helpfull)

Errors

If you are getting a error, please copy and paste the error in the email.

Please direct server related questions and problems to your hosting provider.

Thanks again for your purchase, hope you enjoy using Horizon.
