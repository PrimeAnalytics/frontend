Portfolio

The portfolio uses the isotope jquery plugin.

Documentation

For examples of portfolio page types see the portfolio-* pages.

For ajax independent portfolio pages see the port-single-*.html pages.

For ajax portfolio pages see the port-ajax-* pages.

Filter menu

List your categories in the filter menu in the data-cat attribute. Within the 'li' is what the menu shows, which doesn't have to match the data-cat.

<ul class="filter-menu">
        <li class="selected" data-cat="*">ALL</li> 
    </ul><!-- / .filter-menu -->

Then you can add your items to a category using the data-cat attribute for the portfolio item.

<div class="port-item" data-cat="[CAT]">

A portfolio thumbail

Each portfolio item is wrapped in the .port-item div:

  <div class="port-item" data-cat="[CATEGORY HERE]"><!-- Category -->
    <img src="[THUMBNAIL URL].jpg" alt="Porfolio Image"><!-- portfolio thumbnail --> 
    <a href="[LINK TO SINGLE PROJECT].html" class="port-overlay"><!-- Link to portfolio page -->
      <span class="inner">
        <i class="fa fa-plus-circle"></i>
        <h4 class="title">[Your Project Title]</h4>
        <span>[A little about your project]</span>
      </span>
    </a><!-- End .port-overlay -->
</div><!-- End .port-item-->

A lightbox item

PrettyPhoto Documentation

<div class="port-item" data-cat="[CATEGORY HERE]"><!-- Category -->
  <img src="[THUMBNAIL URL].jpg" alt="Porfolio Image"><!-- portfolio thumbnail -->
  <a class="lightbox-btn" data-pp="prettyPhoto"  href="[LARGE IMAGE FOR LIGHT BOX URL].jpg">
    <span class="inner">
      <i class="fa fa-search"></i>
      <h4 class="title">A Lightbox Project</h4>
      <span>A little about your project</span>
    </span>
  </a><!-- End .lightbox-btn -->
</div><!-- End .port-item-->

To put the lightbox item in a group (for arrows)

<a class="lightbox-btn" data-pp="prettyPhoto[group1]"  href="[LARGE IMAGE FOR LIGHT BOX URL].jpg">

the group1 can be anything, just needs to be the same value for every item you want to be in the same group.

Portfolio control

port-* controls the number of items per row. port-2 to port-5

data-show is how meny items will be visible on page load (for use with the more button).

**

<div class="portfolio port-3" data-show="9">

.port-tiles - Changes the layout style to portfolio tiles but does require a different layout so see a portfolio-tiles page for an example.

<div class="portfolio port-tiles port-3" data-show="9">

.port-no-gutter makes the thumb butt up to each other and removes the thin white line between the thumbnails.

<div class="portfolio port-3 port-no-gutter" data-show="9">
