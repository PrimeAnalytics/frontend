Installation

These folders and files are required to get started, upload them to your web server.

These are the required folders and files:

  • css/
  • img/
  • js/
  • php/ (for the contact form only)
  • index.html

Important notes

Mail server

Please note that the contact function require a working PHP mail server to function.

Loading files in chrome without a server

Also please note that chrome as security restrictions when working locally on files without loading them via server, this prevents the ajax portfolio function working. To test this function locally without using a server simply use to a different browser.

Choosing your index.html version

To start choose which index page you would like to use as a template. All the index versions are preceeded with index-

Rename your choice of homepage page to index.html

Replacing images

Placeholders for all images are in the folder img. The easist way to replace them with your own is to just replace the relevent placeholder file with your own.

Large image files will be the biggest impact on the loading times of the page so don't forget to optimise your images.

Sizes

The sizes on the placeholders are what was used for the demo, but the size can be what ever you like, as there are max-widths set. Smaller images will be better for page speed, larger images will look better on retina devices

Page speed

For the best page speed don't forget to optimise your images, these will have the biggest effect on your page speed.

The scripts are provided in full so you can edit them but for deployment you may want to minify them for the best loading times.

Logos

To just use text you can delete the image tag and replace with text.

To change the logo-light and/or logo.png depending on your color scheme. Replace with your own.

For a retina logo make the image twice the size you need and set a width or hight. If you have a portrait logo set height, for a landscape logo set width.

The header

  • .hd-dark - For a dark menu
  • .hd-grey - For a grey menu
  • .solid - for non transparent headers
  • data-fixed-height="true" - To turn of height switch on scroll

Examples:

Usage:

<div class="header-top">
<div class="header-top hd-dark">
<div class="header-top hd-grey">
<div class="header-top solid">
<div class="header-top solid" data-fixed-height="true">
